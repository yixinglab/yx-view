# yx-view

> A high quality  UI components Library with Vue.js 2.0.



## Install
```shell
npm install yx-view -S
```

## Quick Start
``` javascript
import Vue from 'vue'
import yx from 'yx-view'

Vue.use(yx)


```
For more information, please refer to [Quick Start](https://github.com/) in our documentation.
